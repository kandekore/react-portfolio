## Description

A portfolio of work, summary of skills and resume, built in React with Bootstrap CSS.

- The Motivation for building this application was To showcase what has been learned during the Unviversity of Birmingham Coding Bootcamp
- The application was built to Cement my knowledge of working with React
- It solves the problem of Sending individual CVs to prospective employers or clients.
- Through the process of building this application I learned How to manage routes, components and states within React.

## Usage
